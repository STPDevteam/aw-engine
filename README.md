# Autonomous Worlds Engine
---

This engine is designed to be used in the [AWE Network](https://www.awenetwork.ai/) and is built on top of the [SGLang](https://github.com/joonspk-research/sglang) framework.


## Installation

Use `python -m pip install -e .` to install the package.
(We gonna release it to pypi soon)

## Feature
- Provide an easy to use environment for LLM agents like [OpenAI Gym](https://www.gymlibrary.dev/index.html) for RL agents.
- Achieve higher performance by **unlocking more parallelism**, **load balancing** and **better cache hit rate**.
